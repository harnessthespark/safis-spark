🎲 Random Word Game

  • Pick any object you can see right now
  • Write whatever comes to mind about it - a memory, a story, a feeling, absolute nonsense
  • There's literally no wrong way to do this
Sometimes the weirdest starting points lead to the best writing. A half-eaten banana has launched entire novels.

🎧 Soundtrack Writing

  • Put on a song - any song
  • Write whatever the music makes you see, feel, or imagine
  • You could describe a scene, write lyrics, or just scribble vibes
Music and writing light up the same creative pathways. Some writers can ONLY write with music on.

🔮 "What If" Game

  • Start with "What if..." and finish it with the wildest thing you can think of
  • What if the ocean could talk? What if colours had flavours? What if you could rewrite one day?
  • Follow wherever it takes you
Every great story started as a "what if." J.K. Rowling's was "what if a boy didn't know he was a wizard?"
